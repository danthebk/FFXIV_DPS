  • Stats
    • Delta - The difference between the gear stats and the character's base stats. At level 80, stats have a base value of 380 (340 for determination) which determines the dps floor. The base values do not have any affect on the calculations. A base 380 direct hit will result in a 0% direct hit rate and 380 skill/spell speed will result in a 2.5 second delay.
  • Materia
    • - adds the value of the selected materia.
    • - subtracts the value of the selected materia.
  • Buttons
    • Archive - Stores the current stats for comparison.
    • Revert - Sets the current stats to the values stored in the archive.
  • Notes
    • Calculations are based on a character at level 80. All base values (indicated by an information value) are different for lower levels.
    • This tool is only useful for comparing the secondary stats on equipment assuming they have the same item level. Generally, an item with a higher item level will almost always result in more DPS.
    • The current calculations do not account for character buffs. Certain stats have different levels of effectiveness when combined with abilities. This functionality can be added in a future update. Examples of how abilities affect calculations:
      • Critical hit has increasing returns with higher values. It increases both the critical hit rate as well as critical hit damage. A Bard's Wanderer's Minuet or a Dragoon's Battle Litany increases the party's critical hit rate by 2% and 10% respectively. Critical hit is more valuable for the duration these buffs are active.
      • Determination and Tenacity have diminishing returns with higher values.
      • Direct hit has diminishing returns with higher values. Since direct hit only affects the rate, each additional percentage point has no effect on an ability that is already a direct hit. A Bard's Army's Paeon and Battle voice increases the party's direct hit rate by $ and
      • Skill speed has increasing returns with higher values. The closer a user's delay is equal to 0, attacks can happen exponentially more frequently. Abilities such as Ninja's Huton, Samurai's Shifu, and Monk's Greased Lightning would make skill speed more valuable.

  • Buffs are still experimental and a work in progress. The numbers may not be completely accurate.
